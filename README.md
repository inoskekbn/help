### **Интернет-магазин «CYBER COMBAT»**
Онлайн-платформа для продажи компьютерной техники и геймерского оборудования, созданная с использованием современного стекла технологий. Каждый раздел магазина демонстрирует различные аспекты электронной коммерции.

### **Список разделов и функций**
**Главная страница (index.html)** — Витрина магазина с акционными товарами, новинками и категориями продукции.

**Каталог товаров (встроенный в index.html)** — Полный ассортимент с фильтрацией по категориям, брендам и характеристикам.

**О компании (about.html)** — Информация о миссии магазина, гарантиях и преимуществах.

**Контакты (contacts.html)** — Страница с контактной информацией, формой обратной связи и картой проезда.

### **ДЕТАЛЬНОЕ ОПИСАНИЕ СИСТЕМЫ ЗАКАЗОВ**
**Ключевые возможности**

**Для покупателей:**
- 3 категории товаров с расширенной фильтрацией:
  - Игровые компьютеры
  - Компоненты и апгрейд (видеокарты, процессоры, ОЗУ)
  - Периферия и аксессуары (клавиатуры, мыши, мониторы)
- Динамическое взаимодействие:
  - Добавление товаров в корзину
  - Сравнение характеристик
- Адаптивный дизайн для покупок с любых устройств

**Для администраторов (через панель управления):**
- Управление товарной базой через orders.json
- Просмотр и обработка заказов
- Обновление статусов доставки
- Управление акциями и скидками

**Технические требования и запуск**
Следуйте этим шагам, чтобы запустить магазин на вашем компьютере.

**Минимальные требования:**
- Node.js 16+ (для backend части)
- Современный браузер (Chrome 90+, Firefox 88+)
- Доступ в интернет (для загрузки зависимостей)

**Пошаговая инструкция**
1. **Клонируйте или скачайте проект:**
   - Получите все файлы из структуры ниже

2. **Установите зависимости:**
   ```bash
   cd cyber-combat-shop
   npm install
   ```

3. **Запустите сервер разработки:**
   ```bash
   node server.js
   ```

4. **Откройте браузер:** Магазин будет доступен по адресу `http://localhost:3000`

### **Описание файловой структуры**
**HTML-страницы (навигация магазина)**
- **index.html** — Главная страница с каталогом товаров, фильтрами, корзиной
- **about.html** — Информация о компании, гарантиях, условиях доставки
- **contacts.html** — Контактные данные, форма обратной связи

**Серверная часть (backend/)**
- **server.js** — Express-сервер с API эндпоинтами:
  - GET /api/products — получение списка товаров
  - POST /api/orders — создание нового заказа
  - GET /api/orders/:id — получение статуса заказа
- **orders.json** — JSON-база заказов (товары, клиенты, статусы)
- **package.json** — Зависимости: Express, Cors, Nodemon для разработки

**Клиентская часть (frontend/)**
- **script.js** — Вся бизнес-логика магазина:
  - Загрузка и отображение товаров
  - Работа с корзиной (добавление/удаление)
  - Фильтрация и сортировка товаров
  - Оформление заказа и отправка на сервер
  - Валидация форм
- **style.css** — Стили в тематике киберпанк:
  - Неоновые цвета (синий, фиолетовый, зеленый)
  - Адаптивная сетка товаров
  - Анимации hover-эффектов


**Медиа-ресурсы (img/)**
- **products/** — Каталогизированные изображения товаров
- **Логотипы и брендинг** — Фирменный стиль магазина
