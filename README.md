# GamePC Store - Интернет-магазин игровых компьютеров

## Описание проекта

**CYBERPC** — это современный и удобный веб-сайт для продажи готовых игровых компьютеров и отдельных комплектующих. Проект предназначен для геймеров и энтузиастов, которые хотят быстро и легко собрать или приобрести мощный ПК для игр.

**Основные функции:**
*   **Каталог товаров:** Просмотр готовых сборок и комплектующих (процессоры, видеокарты, ОЗУ и т.д.) с фильтрацией и поиском.
*   **Корзина покупок:** Добавление товаров в корзину, изменение количества и оформление заказа.
*   **Система сборки ПК:** Интерактивный конструктор для выбора совместимых комплектующих.
*   **Адаптивный дизайн:** Удобный просмотр на любом устройстве (ПК, планшет, смартфон).

## Установка и запуск

Этот проект является статическим веб-сайтом. Чтобы запустить его локально, выполните следующие шаги:

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/[inodkekbn]/cyber.pc.git
    ```
2.  **Перейдите в директорию проекта:**
    ```bash
    cd cyber-pc-store
    ```
3.  **Запустите локальный сервер:**
    *   Самый простой способ — использовать встроенный в visual studio терминал:
       

4.  **Откройте браузер** и перейдите по адресу: `http://localhost:8000`
